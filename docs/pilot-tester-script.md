# MathPath Pilot Tester Script

Use this script for the first 3-5 Math Recovery Sprint families. The goal is not a broad launch; the goal is proof that parents understand the gap, students complete early sessions, and the report makes the next step obvious.

## Who To Invite

- Grades 4-8
- Parent has a clear math concern
- Student can complete a short diagnostic independently or with light support
- Family can try 3 sessions in the first week

## Invite Message

Hi [Name], we are testing a 2-week Math Recovery Sprint for students who need a clearer math plan.

The first step is a short diagnostic. It is not a grade. It helps identify the first skill gap to work on, then creates a focused learning path and parent report.

Start here: [Sprint Link]

Would you be open to trying the diagnostic this week and sharing quick feedback after the report?

## Parent Instructions

1. Open the Sprint link.
2. Create or use a parent account.
3. Add one student.
4. Have the student complete the diagnostic.
5. Start the first lesson.
6. Use the tutor if the student gets stuck.
7. Review the Sprint report.
8. Submit the short parent feedback form.

## Operator Checkpoints

- Day 0: invite sent
- Day 1: diagnostic started
- Day 2: diagnostic completed
- Day 3: first lesson started
- Day 7: three sessions completed
- Day 14: report viewed and feedback submitted

Use `/admin/pilot` each day as the action queue. Each family has a priority, due window, and ready-to-send message for the next follow-up.

## Feedback Questions

- Was the report clear?
- Would you continue the sprint?
- What math concern brought you here?
- What sentence would you use to describe the value?

## Success Criteria

- At least 3 families complete the diagnostic
- At least 2 families complete 3 sessions
- At least 2 parents view the report
- At least 2 parents submit feedback
- At least 1 parent gives a usable proof quote

## Follow-Up Message

Hi [Name], thank you for trying MathPath.

I saw that [Student] worked on [Focus Skill]. The report should show the first gap, progress so far, and the next recommended step.

Could you reply with whether the report felt clear and whether you would continue the sprint?
